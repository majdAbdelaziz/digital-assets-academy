# أكاديمية الأصول الرقمية — Digital Assets Academy

موقع تعليمي تفاعلي لتمكين المستثمرين معرفياً ضد الاحتيال الرقمي.

## هيكل الملفات

```
academy/
├── index.html              ← الصفحة الرئيسية
├── join.html                ← انضم إلينا (فورم متصل بـ Google Sheets)
├── simulator.html           ← محاكي النصب التفاعلي
├── modules.html              ← قائمة المحاور التعليمية
├── blog.html                 ← المدونة (6 مقالات)
└── modules/
    ├── persuasion.html       ← المحور 1: هندسة الإقناع
    ├── blockchain.html        ← المحور 2: أساسيات البلوكشين
    ├── wallets.html            ← المحور 3: تأمين المحافظ
    └── regulations.html        ← المحور 4: التشريعات الدولية
```

## الرفع على GitHub Pages

1. أنشئ Repository جديد على github.com
2. ارفع جميع الملفات مع الحفاظ على هيكل المجلدات (مجلد modules منفصل)
3. لرفع مجلد modules: أنشئ ملف جديد واكتب الاسم كـ `modules/persuasion.html` وGitHub ينشئ المجلد تلقائياً
4. اذهب لـ Settings → Pages → Source: main branch
5. انتظر دقيقة — موقعك جاهز

## ملاحظات مهمة

- فورم "انضم إلينا" متصل بـ Google Sheets فعلياً عبر Apps Script — لا تحذف `scriptURL`
- جميع الصفحات مرتبطة بمركز التحقق (verification-hub.net) في الفوتر
